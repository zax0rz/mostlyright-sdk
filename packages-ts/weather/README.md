# @mostlyrightmd/weather

Weather fetchers and parsers for the [mostlyright](https://github.com/helloiamvu/tradewinds) TypeScript SDK — AWC, IEM ASOS, IEM CLI, GHCNh, NWS CLI plus the local-first cache layer. Mirrors the Python `mostlyright-weather` distribution. Declares `@mostlyrightmd/core` as a peer dependency. The current scaffold ships only a placeholder hello function; real implementation lands in TS-W1 onward.
