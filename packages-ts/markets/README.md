# @tradewinds/markets

Prediction-market resolvers for the [tradewinds](https://github.com/helloiamvu/tradewinds) TypeScript SDK — Kalshi NHIGH/NLOW resolvers and Polymarket discover/settle helpers. Mirrors the Python `tradewinds-markets` distribution. Declares `@tradewinds/core` as a peer dependency. The current scaffold ships only a placeholder hello function; real implementation lands in TS-W1 (Kalshi resolvers) and TS-W5 (Polymarket).
